Tech Stack
🐍
Python 3.10+ · Flask 3.x
Core language and web framework. Flask-JWT-Extended for auth, bcrypt for password hashing, Flask-CORS for API access.
🤖
scikit-learn · XGBoost · Optuna
RF, GB, LR, StackingClassifier from sklearn. XGBoost with Optuna hyperparameter search over 100 trials.
🔬
SHAP Explainability
TreeExplainer generates per-prediction SHAP values. Top 6 features shown as a bar chart on every result page.
🗄️
SQLite · APScheduler
predictions.db with users, predictions, drift_log tables. APScheduler sends 30-day email reminders via smtplib.
🔔
Push Notifications · PWA
pywebpush + VAPID for browser push. Service worker enables offline use and PWA installation on mobile.
🧠
TensorFlow LSTM
3-layer LSTM for deep learning comparison. Underperforms ensemble on this dataset size — included for completeness.

React Native wrapper. Biometric auth, daily usage reminders, weekly streak tracking for behaviour change support.
⚙️ Setup Guide

How to Run the Project

Complete step-by-step guide. Follow in order — all 7 steps. The dashboard only works when Flask is running.

First-Time Setup
1
Move dashboard_live.html into templates/
move dashboard_live.html templates\dashboard_live.html
Windows CMD. On Mac/Linux: mv dashboard_live.html templates/
Flask's render_template() can ONLY find files inside the templates/ folder.
2
Add /dashboard route to 04_app_improved.py
@app.route('/dashboard')
def dashboard():
    return render_template('dashboard_live.html')
Paste this anywhere after the imports in 04_app_improved.py. This creates the URL that serves the dashboard.
3
Create and activate a virtual environment
python -m venv venv
venv\Scripts\activate
Mac/Linux: source venv/bin/activate. Isolates your project dependencies.
4
Install all dependencies
pip install flask flask-jwt-extended flask-cors flask-bcrypt xgboost scikit-learn shap optuna tensorflow pywebpush apscheduler pandas numpy
Or: pip install -r requirements.txt
5
Run the Flask server
python 04_app_improved.py
You should see: Running on http://127.0.0.1:5000. Keep this terminal open.
6
Open the dashboard
http://127.0.0.1:5000/dashboard
Opens dashboard_live.html through Flask. Connects to /api/stats and shows live data from predictions.db.
7
Open the assessment form
http://127.0.0.1:5000
index.html is served at root. Students fill the 13-question form — predictions are stored in predictions.db and appear in the dashboard.
